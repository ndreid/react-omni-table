# react-omni-table

Dynamic and multipurpose table for react.

Includes: cross-table drag and drop, cell editing, sorting, row hierarchies, and more

## Installation

Install using [npm](https://www.npmjs.com/get-npm)

```sh
npm install react-omni-table
```
